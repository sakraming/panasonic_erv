# Panasonic ERV Home Assistant 集成

这是一个用于 Home Assistant 的自定义集成，用来控制通过有人模块透传的松下新风系统（ERV）。

## 功能

- 以 `fan` 实体形式在 Home Assistant 中控制新风系统
- 支持开关机
- 支持三档风量：低 / 中 / 高
- 支持两种模式：热交换 / 普通模式
- 读取设备状态并显示更多属性
  - 当前风量档位
  - 当前模式
  - 实际风量档位
  - 实际模式
  - 室内湿度
  - 温度
  - 滤网清扫剩余时间
  - 滤网更换剩余时间
  - 故障代码

## 安装方式

### 方式一：HACS 安装

1. 将本仓库发布为 GitHub 仓库
2. 创建版本标签，例如 `v0.1.0`
3. 在 GitHub 上创建 Release
4. 在 Home Assistant 的 HACS 中添加自定义仓库
5. 选择类型为 `Integration`
6. 安装后重启 Home Assistant

### 方式二：手动安装

将 `custom_components/panasonic_erv/` 目录复制到你的 Home Assistant 配置目录下：

```text
config/custom_components/panasonic_erv/
```

然后重启 Home Assistant。

## 配置

在 Home Assistant 中：

- 打开 `设置`
- 打开 `设备与服务`
- 点击 `添加集成`
- 搜索 `Panasonic ERV`
- 输入有人模块的地址
  - IP 地址，例如 `192.168.30.135`
  - 端口，例如 `8899`

## 设备协议说明

本集成基于你已经验证可用的 Modbus 透传逻辑实现。

### 控制寄存器

- `1`：电源开关
  - `0` = 关机
  - `1` = 开机
- `2`：运行模式
  - `0` = 热交换
  - `1` = 普通模式
- `3`：风量
  - `1` = 低
  - `2` = 中
  - `3` = 高

### 状态寄存器

- `1`：电源状态
- `2`：模式
- `3`：风量
- `4`：实际模式
- `5`：实际风量
- `7`：室内回风湿度
- `8`：温度原始值
- `10`：滤网清扫剩余时间
- `12`：滤网更换剩余时间
- `15`：故障代码

## 温度换算

测试代码中读取到的温度值需要减去 30 才是实际温度：

```text
实际温度 = 原始值 - 30
```

## 已知限制

- 当前集成基于你现有测试代码推断实现，寄存器含义仍建议在实际设备上继续确认
- 风量使用 Home Assistant 的百分比接口映射到三档风量，因此界面上会显示为百分比控制
- 目前只实现了一个 `fan` 实体
